Pyxie Bob

    Pyxie Bob is a brand new programmable device that gets you writing computer code. Simply write your code or edit other peoples and download it to your PyxieBob. Watch the LEDs light up and your PyxieBob come to life.

    Now you can start getting really creative, share code and amaze your friends. You can show off everything you create!

23 Jan 2015 - Windows 8 Installation

    Hi! Some people have asked for help in getting the Pyxie Bob loader running on their Windows 8 machines. To help with this we've created an installer page, with screenshots, showing you precisely how we get the pyxiebob software installed on Windows 8. If you need a helping hand, please take a look!
